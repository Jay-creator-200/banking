# Custom Hooks Directory

This directory contains reusable custom React hooks for frontend state management, data fetching, and browser event listeners.

## Planned Custom Hooks:
- `useAuth()`: Session management wrapper.
- `useActiveBranch()`: Tracks and updates currently selected banking branch for multi-branch operations.
- `useDebounce()`: Input debouncing helper for real-time search inputs.
- `useConfirmDialog()`: Programmatically triggers confirmation dialog before destructive actions.
