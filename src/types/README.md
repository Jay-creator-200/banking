# JSDoc Types Directory

This directory is used to define common JSDoc structures and definitions for our JavaScript files. Since the project uses vanilla JavaScript, JSDoc helps IDEs provide rich autocomplete features and type-checking capability.

## Planned Structures:
- **Member**: Standard customer profile details.
- **Account**: Savings/Deposit/Loan balance structures.
- **Transaction**: Ledger-affecting double-entry records.
- **AuditTrail**: Security logging properties.
