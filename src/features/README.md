# Features Directory

This directory is designed to house self-contained feature modules in future phases of the application. 
Each feature will contain its own components, actions, hooks, and services if they are specific to that feature, following a domain-driven structure.

## Future Banking Modules to be placed here:
- **member-management**: Handle membership registration, KYC, and profile actions.
- **savings-accounts**: Manage savings accounts, deposits, interest credits, and closures.
- **loan-management**: Handle loan applications, credit scoring, underwriting, disbursements, and repayments.
- **teller-operations**: Cash deposit/withdrawal, vault transfers, and teller drawer balance controls.
- **accounting**: General ledger, double-entry journals, chart of accounts, and financial periods.
- **approval-workflows**: Multi-level transaction and document approval workflows.
- **reporting**: Financial reports, balance sheet, trial balance, and regulatory filings.
